This is the ultimate pdf / image merger.
App uses pdf-lib js library.
App is made in ReactJs

Features:-

1.  App merges the pdfs and can also rearrange them in live window.
2.  This app is extremely fast.
3.  This is completely frontEnd application. Which means the servers does not save or know about your data. Very secure.
4.  This can convert JPG/PNG images to PDF.
5.  You can mix JPG file with pdf files. No hassle. It will result in clean A4 size PDF merged pages.
6.  The output size is always A4 (Potrait or landscape), no matter what the input size is.
